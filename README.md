# Mesaidefteri - Giriş Sayfası

Modern, şık ve mobil uyumlu bir giriş sayfası.

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Arka plan görselini `public/background.jpg` olarak ekleyin.

3. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

4. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- Glassmorphism tasarım
